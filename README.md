# SmartWaste Management System (SWMS)

A responsive, enterprise-grade Smart City Waste Management dashboard built with **HTML, CSS, and JavaScript** — no frameworks required.

## 🚀 Features

- **Login Page** — Role selector (Admin, Ops, Dispatcher, Worker), email/password form
- **Dashboard** — Live OpenStreetMap with bin status markers, real-time stat cards, full bin alerts, charts
- **Bin Monitoring** — Searchable bin table with clickable sensor detail panel (fill %, temp, battery, odor, 24H history)
- **Alert Management** — Tabbed alert cards (Full / Near Full / Complaints), assign worker, hotspot map
- **Reports & Analytics** — Daily/monthly charts, top collectors leaderboard, waste concentration map

## 📱 Responsive

Fully responsive across:
- 🖥 Desktop (1400px+)
- 💻 Laptop (1100px–1399px)
- 📱 Tablet (767px)
- 📱 Mobile (390px) — hamburger menu with off-canvas sidebar

## 🗂 Files

| File | Purpose |
|---|---|
| `index.html` | Login page |
| `dashboard.html` | Main SPA dashboard |
| `style.css` | Global design system & responsive CSS |
| `app.js` | Navigation, maps (Leaflet.js), charts, toasts |

## ▶️ How to Run

Just open `index.html` in any modern browser — no server needed!

## 🏙 Tech Stack

- Pure HTML5, CSS3, JavaScript (ES6)
- [Leaflet.js](https://leafletjs.com/) for live maps
- [OpenStreetMap](https://openstreetmap.org/) tiles
- [Google Fonts — Inter](https://fonts.google.com/specimen/Inter)
